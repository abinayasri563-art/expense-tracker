# 💰 Expense Tracker Web Application

## 📌 Project Description
This is a web-based expense tracker built using Flask and SQLite. It helps users manage their daily expenses with login, add, edit, delete, and chart features.

## 🚀 Features
- User Login & Register
- Add / Edit / Delete Expenses
- Monthly Filter
- Chart Visualization
- Total Expense Calculation

## 🛠️ Technologies
- Python (Flask)
- SQLite
- HTML, CSS
- Chart.js

## ▶️ Run Project
1. Install:
pip install flask werkzeug

2. Run:
python app.py

3. Open:
http://127.0.0.1:5000

## 🔐 Security
